# 吉米敢说

学习wepy 的制作的软件，后期做完应该会上线

## 使用

```language: javascript
git clone https://github.com/g1mi/JMGS.git your-path
cd your-path
npm i  //等待安装完所需要的包
npm run dev
上面步骤完成后用微信开发者工具导入目录下dist文件夹即可
```
